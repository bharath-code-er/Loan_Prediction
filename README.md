# 💰 Loan Approval Prediction App

This is a Machine Learning web application built using **Streamlit** that predicts whether a loan application will be approved or not based on user input.

---

## 🚀 Features

- Simple and user-friendly interface
- Real-time loan prediction
- Input fields for applicant details
- Machine Learning model integration
- Interactive UI with sidebar and sliders

---

## 🛠️ Technologies Used

- Python 🐍
- Streamlit
- NumPy
- Joblib (for loading trained model)
- Machine Learning (Classification Model)

---

## 📂 Project Structure


├── app.py # Main Streamlit application
├── loan_model.pkl # Trained ML model
